# Blocking VS Non-Blocking

* 어떻게 프로그램이 작업을 처리하고 완료되는지에 대한 접근 방식을 나타냄
* Blocking과 Non-blocking은 동시성과 관련된 용어로 사용

1. **Blocking (차단) 방식:**
   * **동작 방식:** 한 작업이 수행 중일 때 다른 작업들은 대기
2. **Non-blocking (비차단) 방식:**
   * **동작 방식:** 작업이 수행 중일 때도 다른 작업들을 진행

* Blocking 에서는 한 사용자의 요청을 처리하는 동안 다른 사용자의 요청을 처리하지 못함
* Non-Blocking 방식에서는 한 사용자의 요청을 처리하는 동안에도 다른 사용자의 요청을 동시에 처리
