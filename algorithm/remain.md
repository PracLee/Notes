# 나머지 연산 분배의 법칙
- 덧셈의 분배 법칙 성립 -> (A + B) % C = (A % C + B % C) % C
```python
(20 + 6) % 3 = 26 % 3 = 2
(20 % 3 + 6 % 3) % 3 = (2 + 0) % 3 = 2
```
- 뺄셈의 분배 법칙 성립 -> (A - B) % C = (A % C - B % C) % C
```python
(20 - 6) % 3 = 14 % 3 = 2
(20 % 3 - 6 % 3) % 3 = (2 - 0) % 3 = 2
```
- 곱셈의 분배 법칙 성립 -> (A * B) % C = (A % C * B % C) % C
```python
(20 * 6) % 3 = 120 % 3 = 0
(20 % 3) * (6 % 3) % 3 = (2 * 0) % 3 = 0
```
- 나눗셈의 분배 법칙은 성립하지 않음

- 1부터 100000까지 곱한 값을 1000000007로 나눈 나머지 구하기
```python
import time

MOD = 1000000007
answer = 1
start = time.time()
for i in range(1, 100001):
    answer = (answer * i) % MOD
end = time.time()
print("결과:", answer)
print("수행 시간: {:.6f}초". format(end - start))
```
- 결과: 457992974
- 수행 시간: 0.008505
